How to Play

How to play:

Sudoku is a puzzle game on a 9x9 grid. The goal is to fill every cell with a number (1 to 9) so that no number is repeated in the same row, column, and 3x3 subgrid.

Controls:

Movement:

  • Mouse left-click
  • WASD
  • HJKL
  • Arrow keys

Editing:

  • Numbers 1 to 9.
  • E or ENTER to toggle note mode.
  • 0, SPACE, or BACKSPACE to clear a cell.

Note mode allows you to note multiple possible numbers for a cell.

Press 'SHIFT + E' to automatically generate notes.
